# Code Of Conduct

This project is committed to a respectful and professional collaboration environment.

## Expected Behavior

- Be direct, constructive, and respectful.
- Assume good intent, but prioritize technical accuracy.
- Keep feedback focused on the contribution, evidence, and user impact.
- Welcome contributors with different Autodesk API backgrounds and experience levels.

## Unacceptable Behavior

- Harassment, threats, personal attacks, or discriminatory language.
- Publishing private credentials, customer data, or confidential project information.
- Repeatedly derailing technical discussions.
- Misrepresenting Autodesk documentation, licensing, or affiliation.

## Enforcement

Maintainers may edit, close, or remove issues, pull requests, comments, or contributions that violate this code of conduct. Serious or repeated violations may result in a ban from participating in the project.

To report a conduct issue, open a private maintainer communication channel when available, or contact the repository owner directly.
