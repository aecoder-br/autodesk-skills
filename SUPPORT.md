# Support

Use GitHub issues for:

- Bug reports about a skill.
- Requests for new Autodesk API skills.
- Outdated Autodesk API guidance.
- Broken links or unclear examples.
- Suggestions for repository structure.

This project is maintained on a best-effort basis and does not provide official Autodesk support. For production incidents, account questions, quota changes, billing, or service availability, use Autodesk's official support channels.

When asking for help, include:

- The skill name.
- The Autodesk API or product involved.
- Relevant versions, engines, or endpoints.
- A minimal public example when possible.
- Links to official documentation you already checked.
